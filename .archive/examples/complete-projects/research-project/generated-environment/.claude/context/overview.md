# Climate Change Impact on Coastal Ecosystems Research

## Research Purpose
Systematic literature review and meta-analysis on climate change impacts on coastal ecosystem biodiversity (1970-2020), focusing on temperature rise, ocean acidification, and sea-level changes.

## Primary Research Question
How has climate change affected biodiversity in coastal ecosystems globally from 1970-2020?

## Methodology Overview

### Literature Review
- **Databases**: Web of Science, Scopus, PubMed, Google Scholar
- **Search Strategy**: Systematic queries with defined inclusion/exclusion criteria
- **Quality Assessment**: PRISMA 2020 guidelines
- **Time Period**: Publications from 1970-2024

### Statistical Analysis
- Meta-analysis using random effects model
- Regression analysis for trends
- Time-series analysis
- Spatial analysis with GIS
- Tools: R (meta, metafor), Python (scipy, pandas), QGIS

## Research Phases

### Phase 1: Literature Search (Months 1-2)
Systematic database queries and initial screening

### Phase 2: Data Extraction (Months 3-4)
Detailed reading and structured data entry

### Phase 3: Analysis (Months 5-6)
Statistical analysis and hypothesis testing

### Phase 4: Writing (Months 7-8)
Manuscript preparation and submission
