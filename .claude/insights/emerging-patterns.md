## Confidence Scoring for Emerging Patterns

### Detection Confidence Levels
- **Very High (90-100%)**: Pattern appears in >10 projects with consistent characteristics
- **High (75-89%)**: Pattern appears in 5-10 projects with similar traits
- **Medium (50-74%)**: Pattern appears in 3-5 projects with variations
- **Low (25-49%)**: Pattern appears in 1-2 projects, needs more data
- **Experimental (<25%)**: Single occurrence, monitoring for recurrence

### Validation Criteria
Before promoting to template candidate:
1. Minimum 5 project instances
2. Consistent pattern characteristics (>70% similarity)
3. Clear differentiation from existing templates
4. Demonstrable efficiency improvement
5. User request or feedback supporting need
