# SIREN Project - Claude Code Configuration

This directory contains configuration files for Claude Code to work effectively with the SIREN (Systemic Investigation into REcycling of Neodymium) project.

## Project Overview
SIREN is an academic research project investigating systemic barriers to NdFeB permanent magnet recycling in the EU-27 using Theory of Constraints methodology.
